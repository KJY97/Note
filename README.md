| ------ | ---------------- | -------------------------------- |
| create | folder 최초 생성 | create [과목명] folder           |
| add    | file 최초 생성   | add [과목명] img 파일이름        |
| update | 자료 수정        | update [과목명] 파일이름         |
| delete | 자료 삭제        | delete [과목명] 파일이름         |
| rename | 자료명 수정      | rename [과목명] 수정전 -> 수정후 |
| docs   | README.md 수정   | docs [과목명] readme.md 수정     |
